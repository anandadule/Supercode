# Supercode

Supercode is an AI-powered full-stack web development environment that runs in
the browser. It is a fork of the open-source
[`bolt.diy`](https://github.com/stackblitz-labs/bolt.diy) project by StackBlitz
Labs, retitled, rebranded, and maintained by Anand Adule.

## Goals

- Provide a self-hostable AI development environment with multi-provider LLM
  support (Anthropic, OpenAI, Google, Mistral, Cohere, DeepSeek, Groq, Together,
  Fireworks, Cerebras, OpenRouter, Ollama, LMStudio, and others).
- Integrate with deployment targets (Vercel, Netlify, AWS Bedrock, Supabase)
  and source-control providers (GitHub, GitLab).
- Ship as a wrapped WebContainer app and as an Electron desktop binary for
  Windows, macOS, and Linux.

## Status

This is a personal fork driven first by the maintainer's own needs. Pull
requests are welcome, but there is no formalized epic/feature workflow — see
`README.md` for setup, `CHANGES.md` for a curated change log, and
`CONTRIBUTING.md` for contribution expectations.

## Provenance

Supercode is built on the shoulders of bolt.diy and its many contributors. A
